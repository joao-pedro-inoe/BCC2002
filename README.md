## Nome do Sistema: 
 **PharmaStock** - Sistema de Gestão Farmacêutica 

## Descrição do Sistema: 
 Sistema de banco de dados projetado para o controle rigoroso de estoque de medicamentos em uma farmácia, gerenciando entradas e saídas por lote, validades, rastreabilidade de medicamentos controlados (retenção de receitas) e histórico de vendas.
