**RF01**: O sistema deve registrar os medicamentos separados por lote e data de validade.

**RF02**: O sistema deve permitir o cadastro de fornecedores e fabricantes para cada lote de medicamento recebido.

**RF03**: O sistema deve registrar todas as movimentações de saída, identificando se foi uma venda ou um descarte por vencimento.

**RF04**: O sistema deve vincular a venda de medicamentos de uso controlado (tarja preta/vermelha) a uma receita médica e ao médico prescritor.

**RF05**: O sistema deve registrar o histórico de vendas realizadas, vinculando-as ao funcionário (caixa/farmacêutico) e, opcionalmente, ao cliente.
